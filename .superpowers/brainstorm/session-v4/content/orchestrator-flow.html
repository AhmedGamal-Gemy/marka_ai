Orchestrator Agent: v1 Intent Flow

Intent-based routing strategy for initial stability

1. Intent Parsing

The Orchestrator receives the user message and first categorizes it into one of the following intents:

  • GREETING: Casual chat or help request.
  • GENERATE_POST: Explicit request to create a marketing post.
  • BRAND_QUERY: Questions about the user's business or products.
  • UNKNOWN: Fallback for ambiguous inputs.
